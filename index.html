Sistema Proclin

Angular + Bootstrap - C# .NET - SQL

Link para o projeto !

O Proclin é um Sistema de gerenciamento de clínicas e hospitalar completo e eficiente para a gestão de instituições de saúde. Com uma interface intuitiva, o Proclin permite uma administração eficaz de todos os aspectos do dia a dia das clínicas e hospitais. Desde o agendamento de consultas e procedimentos médicos até o registro de prontuários eletrônicos dos pacientes, simplificando e automatizando as tarefas administrativas, gerando maior organização e produtividade para a equipe médica e administrativa. Projeto desenvolvido em duas etapas: FrontEnd em Angular e Bootstrap, Back End em C# .NET. Integrado ao SQL e hospedado na Onex Data Center.

Sistema Gerenciamento KenzieMED

React - TypeScript - NodeJs - ExpressJs - Vercel

Link para o projeto !

KenzieMED é uma solução inovadora projetada para eliminar a necessidade de marcação de consultas através de intermediários. Com o KenzieMED, os usuários têm o poder de marcar suas próprias consultas diretamente no sistema, de acordo com a disponibilidade de horários. Os profissionais de saúde também se beneficiam do sistema de gerenciamento, pois ele automatiza o processo de marcação de consultas, reduzindo a carga de trabalho administrativo. O projeto foi codado em duas etapas, o Front foi construido em TypeScript, React. Back End foi moldado utilizando Node, Express, TypeORM, integrado ao PostgreSQL e com deploy feito no Railway.

Mercadinho

JavaScript - HTML5 - CSS3

Link para o projeto !

Esté projeto simula um Mercadinho, que codei em JavaScritp, todo o projeto foi estruturado realizando a montagem do DOM e desenvolvi o mesmo para treinar lógica de programação voltada a acrecimos e decrescimos, me pautando em utilizar logicas desestruturadas para realizar os calculos. No layout optei por utilizar um design mais simplista para renderizar os itens e o carrinho, me valendo de CSS puro para dar forma ao site. Basicamente todo conceito de compra, consiste em clicar no item, que será armazenado no carrinho, guardar o valor e concatenar com os demais caso haja, caso o item seja retirado, o preço é decrescido. O Site também conta com um sistema de busca simplificado, onde o produto por ser buscado pelo nome, valor ou tipo.

Proclin System

Proclin is a comprehensive and efficient clinic and hospital management system for the management of healthcare institutions. With an intuitive interface, Proclin allows effective administration of all aspects of the daily operations of clinics and hospitals. From scheduling appointments and medical procedures to recording electronic patient records, simplifying and automating administrative tasks, generating greater organization and productivity for the medical and administrative team. Mercadinho

JavaScript - HTML5 - CSS3

Link to the project!

This project simulates a "Mercadinho" (small market), coded in JavaScript, the entire project was structured by assembling the DOM and developed it to train programming logic focused on additions and subtractions, using unstructured logic to perform calculations. In the layout, I chose to use a more simplistic design to render the items and the cart, using pure CSS to shape the site. Basically, the whole buying concept consists of clicking on the item, which will be stored in the cart, saving the value and concatenating with the others if any, if the item is removed, the price is decreased. The site also has a simplified search system, where the product can be searched by name, value, or type.
