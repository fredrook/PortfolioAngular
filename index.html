Sistema Proclin

Angular + Bootstrap - C# .NET - SQL

Link para o projeto !

Sistema de gerenciamento de clínicas e hospitalar Proclin é uma solução completa e eficiente para a gestão de instituições de saúde. Com uma interface intuitiva e funcionalidades abrangentes, o Proclin permite uma administração eficaz de todos os aspectos do dia a dia das clínicas e hospitais. Desde o agendamento de consultas e procedimentos médicos até o registro de prontuários eletrônicos dos pacientes, o Proclin simplifica e automatiza as tarefas administrativas, proporcionando maior organização e produtividade para a equipe médica e administrativa. É o parceiro ideal para instituições de saúde que buscam um sistema moderno, ágil e eficaz para potencializar seu desempenho e oferecer serviços de excelência.

Sistema de gerenciamento KenzieMED

React - TypeScript - NodeJs - ExpressJs - Heroku

] Link para o projeto !

KenzieMED é uma solução inovadora projetada para eliminar a necessidade de marcação de consultas através de intermediários. Com o KenzieMED, os usuários têm o poder de marcar suas próprias consultas diretamente no sistema, de acordo com a disponibilidade de horários. Através do KenzieMED, os pacientes podem ter um maior controle sobre suas próprias consultas, evitando atrasos e possíveis conflitos de horários. Os profissionais de saúde também se beneficiam do sistema de gerenciamento KenzieMED, pois ele automatiza o processo de marcação de consultas, reduzindo a carga de trabalho administrativo. Isso permite que os médicos e outros profissionais se concentrem no atendimento aos pacientes, proporcionando um serviço de qualidade ainda maior.

Teste Front Stack Sossego

ReactJS + TypeScript

Link para o projeto !

Projeto feito para inserir novos usúarios, com intuito de utilizar somente conceito de context, sem consumo de API's. Baseado nas informações passadas e nas imagens do Figma, realizei uma página de entrada da empresa, que ao clicar, leva ao cadastro do user, em maior analise, notei que o projeto abre margem de interpretação, podendo ou não cadastrar mais de um usúario. Utilizei também lógica de props no styled componets, para rendezição condicional de cores, além de outras lógicas de renderização condicionais como demonstrado no projeto, afim de renderizar modais. Todo o projeto foi feito em ReactJS + TypeScript, utilzando conceitos de Componentização, Pages e Styled Components, Routas e Rota protegida ademais concatenando o uso de ferramentas como Figma, GitHub e libs como React Toast e etc.
